# 🧹 Cleaning Robot Simulator (Python)

A beginner-friendly Python project that simulates a simple cleaning robot. The robot moves through a room, identifies dirty spots, cleans them, and displays the room before and after cleaning.

---

## 📌 Project Overview

This project demonstrates the basic concepts of Python programming by creating a simple cleaning robot simulation.

The robot:
- Scans every position in the room.
- Detects whether a spot is dirty or clean.
- Cleans dirty spots automatically.
- Displays the room before and after cleaning.

This project is ideal for beginners learning Python loops, functions, conditions, and lists.

---

## 🚀 Features

- 🏠 Represents a room using a Python list
- 🤖 Simulates robot movement
- 🧹 Cleans dirty spots automatically
- ✅ Skips already clean spots
- 📊 Displays room status before and after cleaning
- 🎯 Easy-to-understand beginner logic

## ▶️ How It Works

1. Create a room using a list.
2. Each position is marked as:
   - `D` → Dirty
   - `C` → Clean
3. Display the initial room.
4. Robot visits every position.
5. If a position is dirty:
   - Clean it.
6. Otherwise:
   - Skip it.
7. Display the cleaned room.
